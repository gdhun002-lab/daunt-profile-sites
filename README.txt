# daunt individual profile sites

Each folder is a separate deployable website.

Folders:
- hyeeun-gu
- geunyoung-yu
- doyeon-kim

How to deploy on Netlify:
1. Open Netlify.
2. Choose "Add new site" > "Deploy manually".
3. Drag one folder, such as `hyeeun-gu`, into Netlify.
4. Repeat for the other folders.
5. Each folder contains an `index.html`, so it is ready to deploy.

Recommended site names:
- daunt-hyeeun-gu
- daunt-geunyoung-yu
- daunt-doyeon-kim
